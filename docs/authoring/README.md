# 内容作者文档

这一组文档面向主要写脚本、配资源、调配置的人，不需要先读引擎源码。

## 建议阅读顺序

1. [Getting Started](getting-started.md)
2. [脚本语法规范](script-syntax.md)
3. [资源系统与打包](resources.md)
4. [立绘布局 manifest](manifest.md)
5. [config.json 配置说明](config.md)

## 常见任务

- 第一次跑通内容工作流：从 [Getting Started](getting-started.md) 开始。
- 写脚本或查语法边界：看 [脚本语法规范](script-syntax.md)。
- 调整资源路径、fs/zip、打包发布：看 [资源系统与打包](resources.md)。
- 配置立绘站位与预缩放：看 [manifest](manifest.md)。
- 调整入口脚本、窗口、调试配置：看 [config](config.md)。
