# 架构与导航

这里放“先定位再读源码”的入口文档，适合在开始改代码前快速建立范围感。

## 入口文档

- [仓库导航地图](navigation-map.md)：按任务反查 crate、模块和关键文件。

## 什么时候看这里

- 你知道要改行为，但还不知道落在哪个模块。
- 你需要先建立 `vn-runtime` 与 `host` 的边界心智。
- 你想尽量减少大文件源码扫描。
